# SESNtemple

Folder AverageSpec contains all average spectra in [Liu et al. (2016)](http://arxiv.org/abs/1510.08049), and [Modjaz et al. (2016)](http://arxiv.org/abs/1509.07124).

Folder SNIDtemplates contains all SuperNova IDentification ([Blondin & Tonry 2007](http://arxiv.org/abs/0709.4488)) templates in [Liu & Modjaz (2014)](http://arxiv.org/abs/1405.1437), [Liu et al. (2016)](http://arxiv.org/abs/1510.08049), and [Modjaz et al. (2016)](http://arxiv.org/abs/1509.07124). See the README file in the folder for detail.
